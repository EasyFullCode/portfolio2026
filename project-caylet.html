Catálogo digital que reemplazó un PDF de 50 páginas

Diseño y desarrollo de un sitio web con catálogo de productos filtrables, buscador y contacto directo por WhatsApp — convirtiendo un PDF estático en una experiencia de compra navegable para los clientes B2C de Caylet.

Rol UX/UI + Web Dev
Duración 1 mes
Herramientas Figma · Elementor Pro · WooCommerce
Sector Retail
01 — El problema

Un PDF de 50 páginas como única herramienta de ventas.

Cuando un cliente contactaba a Caylet por WhatsApp o redes sociales preguntando por un producto, el equipo de ventas le enviaba un catálogo PDF de 50 páginas. El cliente tenía que desplazarse manualmente por todo el archivo para encontrar lo que buscaba, sin filtros, sin buscador y sin forma de ver detalles o variantes del producto.

Para el equipo de ventas, mantener el PDF actualizado era un proceso lento: cada nuevo producto o cambio de precio requería editar el archivo, exportarlo y volver a compartirlo. No había forma de saber qué productos generaban más interés ni de medir la conversión de consultas a ventas.

50 páginas sin navegación

03 — La solución

Un sitio web que convierte la consulta en contacto directo.

Diseñé y desarrollé un sitio WordPress con WooCommerce como base de catálogo. El cliente puede navegar por categorías, usar el buscador, ver el detalle de cada producto y con un clic abrir WhatsApp con un mensaje pre-llenado. Sin carrito, sin pagos — el objetivo era reducir la fricción al máximo para que el cliente llegue a ventas ya con el producto elegido.

01
Home con propuesta clara
Sección hero con CTA directo al catálogo, categorías destacadas y productos en tendencia para orientar al cliente desde el primer segundo.
02
Catálogo con filtros y buscador
Los productos se pueden filtrar por categoría y buscar por nombre. Sin PDF, sin páginas — el cliente encuentra lo que necesita en segundos.
03
Ficha de producto completa
Fotos, descripción, variantes disponibles y precio. Botón de WhatsApp que abre una conversación con el producto ya mencionado en el mensaje.
04
Admin autónomo
Panel WooCommerce para que el administrador gestione productos, categorías y precios sin tocar código ni depender de un desarrollador.

05 — User flow

Del primer contacto en redes hasta el WhatsApp de ventas.

El flujo cubre el recorrido completo del cliente B2C: desde que descubre la marca en redes sociales o recibe el link de ventas, hasta que hace clic en "Consultar por WhatsApp" con el producto que le interesa. Cada paso está diseñado para reducir la fricción y acortar el tiempo entre el interés y el contacto.

Flujo completo del cliente B2C: descubrimiento → catálogo → producto → WhatsApp

1
Descubrimiento
El cliente encuentra a Caylet en Instagram, TikTok o recibe el link del sitio directamente de ventas por WhatsApp.
2
Llega al home
Ve las categorías destacadas y productos en tendencia. Puede ir directo al catálogo completo o explorar una categoría específica.
3
Navega el catálogo
Filtra por categoría o usa el buscador para encontrar el producto. Ve las tarjetas con foto, nombre y precio de un vistazo.
4
Revisa la ficha del producto
Abre el detalle con fotos ampliadas, descripción completa y variantes disponibles. Toda la información que necesita para decidir.
5
Contacta por WhatsApp
Hace clic en el botón verde. Se abre WhatsApp con un mensaje pre-llenado que incluye el nombre del producto. El equipo de ventas recibe un cliente ya informado.
